# GitHub Workflow Generator

A visual drag-and-drop tool to generate GitHub Actions workflows, built with Flask.

## Features

- **Drag and Drop Interface**: Easily add triggers, jobs, and steps.
- **Visual Configuration**: Configure properties for jobs and steps via a side panel.
- **YAML Generation**: Instantly generate the corresponding GitHub Actions YAML.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python app.py
   ```

3. Open your browser at `http://localhost:5000`.

## Testing

Run the tests with:
```bash
python test_app.py
```
